HTML

HTML stands for Hypertext Markup Language. HTML documents contain text content which describes "what you see" and markup which describes "how it looks". HTML is pretty straight forward and is the majority of documents on the internet. This gives a good overview.

Tags and Elements

HTML documents are made of HTML elements. When writing HTML, we tell browsers the type of each element by using HTML tags. So on this page I made the "tags and Elements" bold above by putting beginning and ending tags to tell the webpage what to do. This gives a good overview.

Inline vs Block Elements

HTML elements are either inline or block. Block elements form an "invisible box" around the content inside of them. Some examples of inline elements are: a, span, br, img, strong. Examples of block elements are: div, form, and p. This gives a good overview.
